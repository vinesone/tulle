TULLE / examples

basic

The three built-in effects over an image or a generated test pattern. Toggle effects to rebuild the pipeline; drag sliders to push uniforms without recompiling.

video

A video element as the source, driven by start(). Falls back to a synthesised clip so the page works with no file on hand.

pointer

The same pointer state reaching an effect two ways — bound straight into a shader as u_pointer, and read from JavaScript via on('pointermove').

custom effect

Vignette and pixelate, defined inline. A fragment shader plus a defaults object is the entire extension API.

composite

Two layers, each with its own source and effect chain, combined by a blend mode with live opacity, then a full-render post chain (grade + vignette). The start of the layout system.

transparency

The same source rendered to a transparent canvas and an opaque one, over a checkerboard — proof that alpha survives the pipeline and the page shows through.

layout

A background with two placed, animated picture-in-picture layers, driven by Transform and setLayerTransform(). The arranger, taking shape.

LUT

Colour grading through a 3D lookup table, built in the browser with makeLut() and applied as a texture — the first effect that samples an auxiliary image.
