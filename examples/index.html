TULLE / examples

WebGL2 effects, composition, text, and animation — no build step, no gl. calls. Start with the playground.
Tulle on GitHub ↗

playground

Build an effect chain live — swap effects with dropdowns, reorder them, drag sliders, load a video or type some text. One-click presets, then take them apart. The whole built-in library, hot-swapped with no reload.

VHS

The wow shot: a generated Outrun scene and an on-screen display, composited and run through a layered analog chain (vhs → scanlines → grade → vignette → grain). Whack the VCR.

explode

Text is a layer source, so a custom shader distorts it. Click to melt and explode the word, click again to reassemble — one eased progress uniform.

layout

Flow layout: sources are boxes that pack inline and wrap until wrapped in a block(); a badge sits absolute. Drag the sliders and watch it re-flow live — solved in design pixels, re-solved every frame, the renderer only sees transforms.

scroll

A scrolling composition — the experience, not a document. Content pans with the wheel, a header stays fixed, and a progress bar + badge are functions of scrollY. Every layer is still live WebGL.

export

Deterministic offline export. An animated scene is walked frame by frame with renderAt(), encoded with WebCodecs, and muxed to .webm — no dependencies. Press Record, get a file.
